## [Minimalist Tech Blog](https://suboptimaleng.github.io) Built with Gridsome && Tailwind CSS

How to get started:
```
git clone https://github.com/SuboptimalEng/suboptimaleng.github.io.git
cd suboptimaleng.github.io
npm install
gridsome develop
```
